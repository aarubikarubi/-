# 🚀 日課ツール (Daily Game Launcher)

**「毎日のゲームの日課、クリック1つで全自動化！」**

複数のゲームを毎日遊ぶの、起動したり閉じたり面倒じゃないですか？
このツールを使えば、1つのゲームを遊び終えて閉じると、**自動で次のゲームが立ち上がります！**

[![Main UI Mockup](https://raw.githubusercontent.com/aarubikarubi/DailyGameLauncher/main/preview.png)](https://raw.githubusercontent.com/aarubikarubi/DailyGameLauncher/main/preview.png)

## ✨ これができる！3つの特徴

* **🎮 閉じるだけで次が起動（連鎖起動）**
  1つ目のゲームを遊び終わって「✕ボタン」で閉じると、設定した秒数後に自動で2つ目のゲームが立ち上がります。
* **🧹 裏の重いランチャーも自動でお掃除**
  ゲームと一緒に開くランチャー（HoYoPlayやSteamなど）も、ゲーム終了時にまとめて自動で閉じてくれます。PCが重くなりません！
* **🎨 綺麗で使いやすいデザイン**
  スマホのアプリみたいに直感的にポチポチ設定できる、見やすくて綺麗な画面です。

## 🛠️ 使い方（たったの3ステップ！）

1. **ダウンロード:** [ここから最新のインストーラー](https://github.com/aarubikarubi/DailyGameLauncher/releases) をダウンロードして、PCに入れます。
2. **ゲームを登録:** ツールの画面で、毎日遊ぶゲームを追加してリストを作ります。
3. **スタート！:** あとは「▶ 日課を開始」ボタンを押すだけ！1つ目のゲームが終われば、あとは勝手に切り替わっていきます。
